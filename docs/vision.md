# ときのきせき — プロダクトビジョン

> あなたの癒しのギャラリーへ、ようこそ

## このアプリは何か

「ときのきせき」は、写真の整理が苦手な人のためのアルバムアプリです。

撮りっぱなしの膨大な写真を、自分のペースで、自分の歩幅で、少しずつ「自分だけのアルバム」に育てていける場所です。

## 解きたい課題

写真はたくさんある。でも整理できない。
見返したい気持ちはある。でも「ちゃんと整理しなきゃ」と思うと手が止まる。

結果、スマホやPCのフォルダに写真が溜まり続け、過去の輝きは埋もれたまま忘れられていく。

## 核となる信念

### 整理は「やらなきゃいけないこと」ではなく、「自然と進んでいくもの」であるべき

自動で完璧に整理してくれるアプリではない。
「今日はちょっとだけ触ろうかな」を積み重ねるうちに、いつの間にかアルバムができている。そういうアプリを作る。

### ゴールは「自分だけのアルバム」。でもそこへの道は一つじゃない

- タイムラインを眺めるだけの日があっていい
- ベストを1枚だけ選ぶ日があっていい
- タグを気まぐれにつける日があっていい
- スライドショーでただ癒される日があっていい

そのすべてが、少しずつアルバムに近づいている。

### 歩幅はユーザーが決める

状況も気分も日によって違う。このアプリは特定の歩幅を強制しない。大きな一歩も、小さな一歩も、立ち止まることも、すべて許容する。

## 「記録」と「記憶」

写真には二つの性質がある。

- **記録**：「ここに行ったよね」という事実の証拠
- **記憶**：「これは残したい」と思う、心が動いた瞬間

撮った時点では、すべてが「記録」。
そこからベストを選び、タグをつけ、見返すうちに、一部が「記憶」に変わっていく。

ときのきせきは、この「記録から記憶への変換」を手伝うアプリである。

## 名前に込めた意味

「ときのきせき」= 時の軌跡。

過去の日々が写真として残っている。それはかつて放たれた星の光のようなもの。遠い昔の輝きが、今ここに届く。ときのきせきは、その軌跡をたどり、もういちど輝かせるための場所。

「きせき」は「奇跡」とも読める。日常の中の小さな輝きを見つけ出す行為は、ささやかな奇跡でもある。

## 設計原則

### 1. 歩幅を強制しない
新機能を追加する時の最も重要な問い：「それはユーザーに特定の歩幅を強制しないか？」
どの機能も「使わなくてもいい」状態を保つ。

### 2. 触れるだけで前に進む
タイムラインを眺めること、1枚にベストをつけること、タグを1つつけること。どんな小さな行動も、アルバム完成に向けた一歩として蓄積される設計にする。

### 3. 癒しを最優先にする
効率的に整理させることが目的ではない。写真を見返して「ふふっ」となる体験が最優先。整理はその副産物。

### 4. 明るく、温かく
UIもトーンも、切なさや重さではなく、明るさと温かさを基調にする。過去を振り返ることは後ろ向きではなく、今を豊かにする行為である。

## ユーザー像

- 写真や動画をたくさん撮る
- 見返したい気持ちはあるが、整理は苦手
- 「ちゃんとやらなきゃ」のプレッシャーで手が止まる
- 推し活、旅行、ペット、イベントなど、見せたい・残したい瞬間がある
- 気分に波があり、やる気のある日とない日の差が大きい

## ロードマップの方向性（MVPの先）

MVPは「フォルダ指定 → イベント一覧 → ベスト選び → スライドショー」の一本道。
ここから先は、以下のような機能で「歩幅の選択肢」を増やしていく：

- **タグ付け**：自由なラベルで写真を横断的にまとめる（HG、犬、旅行…）
- **人物ごとの表示**：特定の人にまつわる写真だけを見る
- **重複検出**：似た写真を提示して選びやすくする（片付けの歩幅を下げる）
- **場所の推定・サジェスト**：「この日は〇〇にいたかも？」を提案してくれる
- **動画対応**：写真と動画が混在するスライドショー
- **テーマ別スライドショー**：ベストだけ、犬だけ、特定の年だけ、など
- **イベントの結合**：連続するイベント日を手動でまとめてアルバム化

いずれの機能も「使っても使わなくてもいい」を守る。
